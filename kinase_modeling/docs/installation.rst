============
Installation
============


Stable release
--------------

To install Kinase Modeling, run this command in your terminal:

.. code-block:: console

    $ pip install kinase_modeling

This is the preferred method to install Kinase Modeling, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Kinase Modeling can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/steven-albanese/kinase_modeling

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/steven-albanese/kinase_modeling/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/steven-albanese/kinase_modeling
.. _tarball: https://github.com/steven-albanese/kinase_modeling/tarball/master
